Detailed Description

The BaseAnnotationHighlighter class is the base class for a highlighter for annotation lines of the form 'changenumber:XXXX'.

The change numbers are assigned a color gradient. Example:


  112: text1 <color 1>
  113: text2 <color 2>
  112: text3 <color 1>
